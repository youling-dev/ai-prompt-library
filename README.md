## 📖 简介

在与 AI 模型交互时，**提示词的质量决定了输出的质量**。这个项目收集了经过验证的高质量提示词模板，涵盖编程、写作、分析、设计等场景。

不只是提示词——还附带一个 CLI 工具，让你在终端快速搜索和使用提示词。

---

## ✨ 功能

- 📋 **18+ 精选提示词模板** — 覆盖开发、写作、分析、学习、效率等场景
- 🖥️ **CLI 工具** — 终端内搜索、复制、使用提示词
- 🏷️ **分类管理** — 按场景、模型、难度分类
- 🔄 **持续更新** — 定期收录新模板

---

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/youling-dev/ai-prompt-library.git
cd ai-prompt-library

# 安装
pip install -r requirements.txt

# 搜索提示词
python cli.py search "code review"

# 查看并复制提示词
python cli.py view "code-review-strict"
```

## 📂 分类

### 💻 编程开发（5 个）
- `code-review-strict` — 严格代码审查
- `debug-assistant` — 调试助手
- `architecture-design` — 架构设计
- `api-design` — API 设计评审
- `docker-compose` — Docker Compose 生成

### ✍️ 写作创作（3 个）
- `technical-writer` — 技术文档写作
- `ai-taste-enhancer` — 提升 AI 输出品味
- `changelog` — Changelog 生成

### 📊 数据分析（2 个）
- `sql-generator` — SQL 生成
- `data-cleaning` — 数据清洗

### 🎓 学习研究（3 个）
- `concept-explainer` — 概念解释
- `paper-summarizer` — 论文总结
- `roadmap` — 学习路线规划

### 🛠️ 效率工具（5 个）
- `regex-helper` — 正则表达式生成
- `bash-helper` — Shell 脚本助手
- `config-generator` — 配置文件生成
- `commit-message` — Git Commit Message
- `standup-report` — 站会报告
- `git-workflow` — Git 工作流设计
