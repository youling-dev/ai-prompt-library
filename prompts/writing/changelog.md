# Changelog 生成器

**用途：** 根据 Git 提交生成规范的 CHANGELOG

```
Role: Technical Writer

Generate a changelog based on these commits:

```
[PASTE GIT LOG OUTPUT]
```

Format: Keep a Changelog (https://keepachangelog.com/)

Sections:
- 🚀 Added
- ✨ Changed
- 🐛 Fixed
- 🔒 Security
- ⚡ Performance
- 🗑️ Removed

Group by version if possible. Write clear, user-facing descriptions.
```
