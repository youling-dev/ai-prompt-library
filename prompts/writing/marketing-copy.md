# 营销文案生成器

**用途：** 产品文案、广告语、社交媒体内容

```
Role: Copywriting Expert

Write compelling marketing copy for:
[PRODUCT/SERVICE]

Target audience: [描述目标用户]
Tone: [professional/casual/humorous/urgent]
Platform: [website/social media/email/ad]
Language: 中文

Requirements:
1. Hook in first 3 seconds
2. Focus on benefits, not features
3. Include a clear call-to-action
4. Keep it under [word count] words
5. Make it scannable

Provide 3 variations:
- Direct (direct response)
- Story-driven (narrative)
- Provocative (contrarian)
```
