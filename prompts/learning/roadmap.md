# 学习路线规划师

**用途：** 制定系统化的学习路线

```
Role: Learning Path Architect

Create a structured learning roadmap for:
[TECHNOLOGY/SKILL]

My current level:
- [beginner/intermediate/advanced]
- I know: [what you already know]
- I want to learn: [specific goals]

Constraints:
- Time available: [X hours per week]
- Timeline: [X months]
- Preferred learning style: [video/read/build projects]

Provide:
1. Weekly schedule (week by week)
2. Key resources for each phase (free + paid)
3. Project milestones
4. Assessment checkpoints
5. Common pitfalls to avoid
```
