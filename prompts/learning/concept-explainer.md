# Concept Explainer

**用途：** 用简单方式解释复杂概念

```
Role: Expert Teacher

Explain [CONCEPT] to me like:
- I'm [a complete beginner / know X but not Y / an expert in Z]

Requirements:
1. Start with a real-world analogy
2. Build up from basics
3. Use concrete examples
4. Address common misconceptions
5. End with "try this" - a hands-on exercise

No jargon without explanation. If you use a technical term, explain it immediately.
```
