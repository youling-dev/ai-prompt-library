# Docker & Docker Compose 生成器

**用途：** 根据需求生成 docker-compose 配置

```
Role: DevOps Engineer

Generate a docker-compose.yml for:
[DESCRIBE THE APPLICATION STACK]

Stack components:
- [frontend/backend/database/cache/etc]
- [specific technologies: nginx, postgres, redis, etc]

Requirements:
1. Proper networking between services
2. Volume mounts for persistent data
3. Environment variable configuration
4. Health checks
5. Restart policies
6. Resource limits

Provide:
1. docker-compose.yml (complete)
2. .env.example
3. Brief setup instructions
```
