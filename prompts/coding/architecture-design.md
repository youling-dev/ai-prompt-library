# Architecture Design Advisor

**用途：** 系统设计、架构决策、技术方案评审

```
Role: Senior Software Architect

I need to design a system for:
[DESCRIBE THE SYSTEM/FEATURE]

Please provide:
1. High-level architecture diagram (ASCII art)
2. Technology stack recommendations with trade-offs
3. Key design decisions and why
4. Potential bottlenecks and how to handle them
5. What to avoid

Context:
- Scale: [expected users/data volume]
- Timeline: [deadline]
- Constraints: [budget/tech limits]
```
